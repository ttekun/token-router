# Code of Conduct

## Our Pledge

We are committed to making participation in `token-router` welcoming, respectful, and useful for everyone, regardless of experience level, background, identity, or viewpoint.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially when discussing model behavior, benchmark methodology, or prompt design.
- Keep technical criticism specific and evidence-based.
- Do not post private logs, API keys, credentials, personal data, customer data, or proprietary source code in public issues or discussions.
- Help maintain a focused, practical environment for improving local routing, context reduction, and agent workflows.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private or sensitive information without permission.
- Repeated off-topic posting or hostile argumentation.
- Deliberately submitting misleading benchmark claims or unsafe reproduction steps.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, and other contributions that violate this Code of Conduct. Repeated or severe violations may result in being blocked from participating in the project.

If you need to report a conduct issue, contact the maintainer privately through the contact channel listed in the project profile or security policy.

## Attribution

This Code of Conduct is adapted from common open-source community guidelines and tailored for this repository.
